# 🛒 E-commerce Customer Analytics

## 📌 Overview
This project analyzes e-commerce customer and sales data using SQL, Python, and Power BI. It demonstrates data cleaning, exploratory data analysis, business insights, and interactive dashboard creation.

---

## 🛠️ Technologies Used
- SQL Server
- Python
- Power BI
- Microsoft Excel

---

## 📂 Dataset
The project uses two datasets:
- ecommerce_customers.csv
- ecommerce_orders.csv

---

## 📊 Project Workflow

### 1. SQL
- Created database and tables
- Imported customer and order datasets
- Performed JOIN operations
- Used GROUP BY and aggregate functions
- Analyzed monthly sales
- Identified top customers
- Category-wise sales analysis
- State-wise revenue analysis
- Calculated average order value

### 2. Python
- Data cleaning using Pandas
- Exploratory Data Analysis (EDA)
- Sales trend visualization
- Product category analysis
- Customer spending analysis
- Revenue insights using charts

### 3. Power BI
Created an interactive dashboard containing:
- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value KPI
- Monthly Sales Trend
- Sales by Category
- Sales by State
- Top Customers
- Interactive Filters (Slicers)

---

## 📈 Key Insights
- Identified the highest revenue-generating product categories.
- Found top customers based on purchase amount.
- Analyzed monthly sales trends.
- Compared revenue across different states.
- Evaluated customer purchasing behavior.
